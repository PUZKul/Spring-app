Pobierz liste książek
/api/library/books

Opis

Metoda ta przyjmuje 3 wymagane paremetry (limit, page, sort) oraz jeden parametr opcjionalny (order)

limit - liczba książek do pobrania (min 1, max 50)
page - numer strony. Nie może być mniejszy od 0
sort - sortowanie po konkretnym polu książki. Dostepne wartości: TITLE, RATING, YEAR, POPULARITY
order - parametr opcjonalny. Określa kierunek sortowania. Wartość domyślna (ASC). Dostepne wartości: ASC, DESC

Przykład

api/library/books?limit=10&page=0&sort=TITLE Pobranie 10 ksiązek z pierwszej strony posortowanych po tytule od A do Z
api/library/books?limit=10&page=1&sort=RATING&order=DESC Pobranie 10 ksiązek z drugiej strony posortowanych po ocenie od najwiekszej

Uwierzytelnienie

Nie wymaga

Typ zapytania

GET

Zwracana wartość

Lista LibraryBook

Wyjątki

  • BAD_REQUEST (400) - najczęściej spowodowane błędnym parametrem
Pobierz książke po ID
/api/library/books/id/{id}

Opis

Metoda zwraca jedną książe o podanym id w parametrze. W przypadku braku książki z podanym id zostaje zwrócony wyjątek

Przykład

api/library/books/id/1 - Pobranie książki o id równym 1.

Uwierzytelnienie

Nie wymaga

Typ zapytania

GET

Zwracana wartość

Optional LibraryBook

Wyjątki

  • BAD_REQUEST (400) - najczęściej spowodowane błędnym parametrem
  • NOT_FOUND (404) - Książka z podanym id nie istnieje w bazie
